# Chinese-Text-Segmentation-and-Word-Frequency-Statistics
dictionary.txt为分词词典，共包含217146个词语。
stoplist.txt为停用词词典，共包含1208个词语。

## 一、目的
 
- 理解中文分词任务的基本概念与意义；
- 掌握正向最大匹配（FMM）、逆向最大匹配（BMM）的原理与实现方法；
- 实现分词，并对结果进行词频统计与排序。
 
## 二、任务
 
- 基于提供的词典，实现正向/逆向最大匹配法的中文分词程序；
- 文本作为测试样例，用程序完成分词、词频统计及排序；
- 输出分词结果、词频统计与排序信息，记录程序运行情况；
- 人工检查分词错误（如歧义切分、未登录词识别错误），记录并分析原因；
